# Speedrun Stylus — Counter

A learning exercise from the **Speedrun Stylus** challenge series: the "counter"
smart-contract task built on **Arbitrum Stylus**.

The exercise lives in the `speedrun_stylus_counter/` directory. See the
[Speedrun Stylus](https://www.speedrunstylus.com/) challenges for the walkthrough
and setup instructions.

## Stack

- Arbitrum Stylus (smart contracts)
- Yarn workspace scaffold
